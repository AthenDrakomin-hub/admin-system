# 核心页面结构总览

## 页面架构

```
/login                          # 登录页
/admin                          # 管理后台
  ├── /dashboard                # 统一待办工作台 ⭐
  ├── /trade                    # 交易审核模块
  │   ├── /a-share              # A股交易审核 ⭐
  │   ├── /hk-share             # 港股交易审核 ⭐
  │   ├── /ipo                  # 新股申购审核 ⭐
  │   ├── /block                # 大宗交易审核 ⭐
  │   └── /board                # 一键打板审核 ⭐
  ├── /user                     # 用户管理模块
  │   ├── /list                 # 用户列表 ⭐
  │   ├── /funds                # 资金上下分 ⭐
  │   ├── /positions            # 持仓管理
  │   ├── /orders               # 订单管控
  │   └── /flows                # 流水管控
  ├── /finance                  # 财务审核模块
  │   ├── /recharge             # 充值审核 ⭐
  │   └── /withdraw             # 提现审核 ⭐
  ├── /system                   # 系统配置模块
  │   ├── /params               # 全局参数配置 ⭐
  │   ├── /admins               # 管理员管理
  │   └── /audit                # 审计日志
  └── /market                   # 行情管理
```

## 核心页面详情

### 1. 统一待办工作台 ✅

**路径**: `/admin/dashboard`

**功能**:
- 待审核交易统计（实时）
- 待审核充值统计（实时）
- 待审核提现统计（实时）
- 异常订单提醒
- 今日数据概览
- 30秒自动刷新

**状态**: ✅ 已实现（含轮询）

---

### 2. A股交易审核 ✅

**路径**: `/admin/trade/a-share`

**功能**:
- 待审核订单列表
- 价格/数量/金额展示
- 手续费计算
- 一键审核通过/驳回
- 订单详情查看

**状态**: ✅ UI已实现

## 技术实现

### UI框架
- Next.js 14 App Router
- Tailwind CSS（银河蓝主题）
- lucide-react 图标

### 数据通信
- REST API
- 30秒轮询（工作台）
- 5秒轮询（行情）

### 状态管理
- React Hooks
- usePolling 自定义Hook

### 样式特点
- 现代化卡片设计
- 渐变色背景
- 响应式布局
- 悬停效果
- 加载状态
